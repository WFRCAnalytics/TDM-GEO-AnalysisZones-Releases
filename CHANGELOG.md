# Changelog

All notable changes to the TAZ (and future MAZ) dataset are documented here.
Releases are tagged in Git as `vYYYY.N` (e.g., `v2024.1`).

## [Unreleased]

## [v9.0.1] - 2024-04-12
### Changed
- Renamed shapefile to WFv901_TAZ_20240412.shp (versioned naming convention).
- Updated TAZ attribute data.

## [v9.0.0] - 2023-07-21
### Changed
- Updated TAZ geometry and attribute data (3,546 features).

## [v8.3.2] - 2021-11-10
### Changed
- Updated TAZ attribute data (TAZ.dbf).

## [v8.3.0] - 2019-09-18
### Added
- Initial versioned release of Wasatch Front TAZ shapefile.
- 2,881 TAZ features in NAD83 UTM Zone 12N (EPSG:26912).
